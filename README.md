# Kuma's Online Shop

## Key Features
- Browse through a wide range of products from different categories.
- Add products to the shopping cart and manage quantities.
- View detailed product information, including images, descriptions, and prices.
  
## Technologies Used
- React.js
- JavaScript
- CSS
- Bootstrap react
